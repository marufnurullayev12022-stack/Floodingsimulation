ASOSIY TAKOMILLASHTIRISHLAR
================================================================================

✅ Manning's Equation
   - Haqiqiy water physics
   - V = (1/n) × R^(2/3) × S^(1/2)
   - Suv tezligini to'g'ri hisoblaydi

✅ Infiltration Modeling
   - Green-Ampt model
   - Suv yo'qolishini simulatsiya qiladi
   - Tuproq samaradorligini hisobga oladi

✅ Velocity Tracking
   - u,v components
   - Magnitude field
   - Flow visualization (yorq ko'k = tez)

✅ CFL Stability
   - Adaptiv timestep
   - Tezroq, barqaror hisoblash
   - 2-5x speedup

✅ Advanced Statistics
   - maxVelocity (hazard assessment)
   - meanVelocity
   - totalVolume
   - infiltratedVolume
   - channelLength

================================================================================
PARAMETRLAR
================================================================================

Manning's Roughness (manningN):
────────────────────────────────
0.01  = Concrete (smooth)
0.03  = Grass (DEFAULT)
0.05  = Dense vegetation
0.08  = Forest
0.10  = Very dense forest

→ Kichik = Tezroq oqim
→ Katta = Sekinroq oqim

Infiltration Rate:
──────────────────
0      = Yo'q (asfalt, beton)
0.00001 = Juda sekin (clay)
0.0001  = Sekin (loam)
0.001   = Tez (sand)

→ Kichik = Suv qoladi (ko'p pool)
→ Katta = Suv yo'qoladi (tez tashqi)

PARAMETRLAR SOZLASHI UCHUN MASLAHAT
================================================================================

MANNING'S N:

1. Birinchi = DEFAULT (0.035)
   → Ko'roq loyihalarda ishlaydi

2. Agar suv TEZROQ KERAK:
   → manningN KAMAYTIR (0.025-0.030)
   → Foyda: More spreading, faster flow

3. Agar suv SEKINROQ KERAK:
   → manningN OSHIR (0.050-0.080)
   → Foyda: More pooling, slower channels

4. Haqiqiy ma'lumotlar bilsangiz:
   → Manning's formula dan orqaga
   → Velocity = (1/n) × R^(2/3) × S^(1/2)

İNFİLTRATSİYA:

1. Birinchi = MINIMAL (0.000001)
   → Ko'prog hazorni shirish uchun

2. Agar SUVI JOY JONIDA yo'q:
   → infiltrationRate OSHIR
   → Foyda: Water disappears quickly

3. Agar FLOOD TO'P QOLSA:
   → infiltrationRate KAMAYTIR
   → Foyda: Water persists longer
